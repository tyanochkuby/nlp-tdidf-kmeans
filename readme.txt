Ten program stosuje metodę uczenia nienadzorowanego k-średnich i macierz tdidf. 
Aby dobrać jak najlepszą liczbę k, do oceniania używam silhouette_score. 
Wypisywane są kilka maili z każdej grupy.
Używam również PCA do redukcji wymiarów i wizualizacji danych.
